wikipedia
  • Graph: A directed graph G(V,E)
  • Supply & Target: supply(S) and a target(T), want to send from supply to target, there can be infinite supply avaliable.
  • Capacity: Each edge have some capacity c(x,y)
  • Flow: The actual amount we send is called flow f(x,y). And it cannot exceed capacity. 0 ≤ c(x,y) ≤ f(x,y).
  • Flow Conservation: For each node that is not supply or target, the incoming flow is equal to outcoming flow.
  • Output: Compute flow on each edge such that we have maximum flow coming to target.
